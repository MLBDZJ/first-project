# 个人网站 - 马路边的座机

这是一个响应式、现代化的程序员个人网站，基于你的自我介绍博客内容构建。

## 文件结构

```
├── index.html          # 主页面
├── style.css           # 样式文件
├── script.js           # 交互脚本
├── 自我介绍_博客.md    # 原始博客内容
└── README_网站.md      # 本说明文件
```

## 如何查看网站

### 方法1：直接打开（最简单）
直接在浏览器中打开 `index.html` 文件。

### 方法2：使用本地服务器（推荐）
```bash
# 使用Python（如果已安装）
python -m http.server 8000

# 或使用Node.js（如果已安装http-server）
npx http-server
```
然后访问 http://localhost:8000

### 方法3：使用VS Code Live Server扩展
如果你使用VS Code，安装"Live Server"扩展，然后在 `index.html` 文件上右键选择 "Open with Live Server"。

## 自定义和扩展

### 修改内容
1. 打开 `index.html` 修改文本内容
2. 在 `style.css` 中调整颜色、字体和布局
3. 在 `script.js` 中添加或修改交互功能

### 主要颜色变量（在style.css中）
```css
:root {
    --primary-color: #00d4ff;      /* 主色调 */
    --primary-dark: #00a8cc;       /* 主色调深色 */
    --secondary-color: #6c63ff;    /* 辅助色 */
    --dark-bg: #0a0a0f;           /* 深色背景 */
    --darker-bg: #07070c;         /* 更深背景 */
    --card-bg: #121218;           /* 卡片背景 */
    --card-border: #1e1e2e;       /* 边框色 */
}
```

### 添加新页面
如果需要添加新页面（如博客文章、作品集），可以：
1. 创建新的HTML文件
2. 在导航栏中添加链接
3. 保持一致的样式

### 部署到线上
1. **GitHub Pages**（免费）
   - 创建GitHub仓库
   - 将文件推送到仓库
   - 在仓库设置中启用GitHub Pages

2. **Vercel/Netlify**（免费，支持自定义域名）
   - 连接Git仓库
   - 自动部署
   - 支持HTTPS

3. **自有服务器**
   - 上传文件到服务器
   - 配置Nginx/Apache

## 技术细节

### 性能考虑
- 最小化HTTP请求（只有一个CSS和一个JS文件）
- 使用CSS变量便于主题切换
- 代码经过优化，无性能瓶颈

### 可访问性
- 语义化HTML标签
- 适当的ARIA属性
- 键盘导航支持
- 足够的颜色对比度

### SEO优化

- 元标签（可在index.html头部添加更多）
- 语义化结构
